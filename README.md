# Nexus Analytics | AI & Power BI Dashboard

**🟢 LIVE DASHBOARD:** [View the Dashboard Online](https://simanta9190-c.github.io/AI-SUSTAINABILITY-DASHBOARD/)

A highly performant, visually premium, lightweight data intelligence dashboard simulating advanced AI Cloud telemetry and Power BI processing metrics. 

## 🚀 Features
*   **Power BI Native Aesthetic:** Employs crisp whites, clean boundaries, Microsoft Blue, and PowerBI Yellow accents mimicking native Microsoft Enterprise tools.
*   **Dynamic Tab Navigation:** Seamless, zero-latency transitions between Execute Summary, Node Compute, Population, and Map Data views.
*   **Advanced Chart.js Visualizations:** Features interactive Polar Area, Bubble Scatter, Radar Comparison, Stacked Bar, and Dual-Axis Mixed charts.
*   **Geospatial Tracking:** Uses `Leaflet.js` to render live server ping constraints globally and across target subcontinents.
*   **Vanilla Stack Execution:** Zero dependency on node or build-tools. Runs purely on optimized HTML/CSS/JS.

## 🛠 Project Structure
- `index.html` — The main structural tabbed interface.
- `style.css` — Robust native UI styling logic and CSS variables.
- `script.js` — All routing intelligence, live data ticking algorithms, and Chart.js initialization logic.

## 🏃‍♂️ How to Run Locally
Because this project utilizes zero heavy backend dependencies, you can launch the dashboard using a basic HTTP text server.
From your terminal, navigate inside the project folder and run:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000` in your browser.
